# AI Agent Architecture

## Core Agents
1. **Discovery Agent** - Analyzes music and suggests targeting
2. **Content Creation Agent** - Generates promo materials
3. **Outreach Agent** - Playlist and PR pitching
4. **Ad Optimization Agent** - Manages paid campaigns
5. **Analytics Agent** - Reports and insights

Orchestrator coordinates the swarm.

## Implementation
Use LangChain or CrewAI with tools for Spotify API, ad platforms, etc.
